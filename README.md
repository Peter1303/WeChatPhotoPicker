# WeChatPhotoPicker
Image Picker like Wechat and Weibo for share photo.
(高仿微信发朋友圈图片选择器、微博发动态，实现APP常见的图文动态分享)

1. 支持拍照
2. 流畅加载数千张图片无OOM
3. 支持多图选择
4. ViewPaper图片预览
5. 分文件夹显示图片
